
# UNIX man pages for the dispersion toolkit

## Build instructions

Using BASH, executing

````
bash transduce.sh
````

generates the man pages to be found in

````
man1/
````

using the source markdown files. This generative step requires 

````
pandoc
````

to be installed on your system.
